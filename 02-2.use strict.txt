'use strict';

순수 Valina JavaScript 사용할 때 맨 윗줄에 정의하면 좋다.
(Typescript는 안써도됨)

왜?
1. JavaScript는 빠른 시간 10일 안에 만든 언어(굉장히 유연한 언어)
    유연하다는 말은 많은 실수를 할 수 있다는 말이기도 하다.
    ECMAScript 5 에서 선언되어있는 use strict 는 
    선언하지 않고 변수를 사용하는 그런 행위를 막아준다.

2. 상식적인 범위 안에서 개발가능, 자바스크립트 엔진이 효율적으로 자바스크립트를 분석하기 때문에
    성능개선 가능!

-> strict 모드로 개발하기를 권장!


// 엄격 모드가 여기서부터 적용됩니다.
"use strict"


// 엄격 모드가 이 파일 전체에 적용됩니다.
'use strict';
